🎉 Novedades v1.2

🔧 Correcciones Críticas

  • HTTP 500 Fix: El directorio de sesiones ahora se crea antes de que Session(app) se inicialice
  • Timeout de Gunicorn: render.yaml ahora inicia gunicorn con --workers 1 --timeout 120
  • Bot 24/7: El polling del bot de Telegram ahora corre en un thread daemon
  • Fix de header de autenticación: Cambio de Authorization: Bearer → X-API-Key
  • Fix de limpieza de keylog: keylog.txt se vacía solo después de un envío exitoso

✨ Nuevas Características

  • Gestor de Actualizaciones: Seguimiento automático de versiones en GitHub
  • Notificaciones de Actualización: Todos los usuarios reciben mensajes en Telegram cuando hay nueva versión
  • Nuevos Comandos del Bot: /help, /version, /key, /download
  • Dashboard Mejorado: Guía de onboarding paso a paso
  • Snippets de Código: Generador para Python, Bash, PowerShell, Node.js, PHP

🔐 Mejoras de Seguridad

  • Bloqueo de Plataformas: Linux, macOS, Android, iOS bloqueados en páginas públicas
  • SESSION_COOKIE_SECURE: Habilitado solo cuando FLASK_ENV=production
  • OWNER_ID: Siempre se añade a authorized_users
  • Validación mejorada: Mejor manejo de errores de autenticación

⚙️ Cambios Técnicos

  • explorer.py: check_for_updates convertido a async def con APScheduler
  • app.py: Mejor integración con Flask-Session y gunicorn
  • render.yaml: Configuración optimizada para producción
  • details.xml: Declaración UTF-8 y formato consistente
